# Workflow: workflow-init

Short description: Initialize a project-specific BMAD workflow for local tasks and onboarding.

## Overview

This workflow provides a minimal, extensible BMAD workflow that can be customized for the repository.

## Files

- `methods.csv` — list of available methods for the workflow.
- `steps/` — contains step-by-step instruction files used by the workflow.

## Steps

1. `step-01-setup.md` — Repo setup and agent loading.
2. `step-02-run.md` — How to run the workflow with BMAD tooling.

## Usage

Follow the instructions in the `steps/` files to run or adapt this workflow for your project.
